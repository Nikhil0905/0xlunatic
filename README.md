# Nikhil Shakya - Cybersecurity Portfolio

Welcome to the source code for my personal cybersecurity portfolio website.

**Live Deployment:** [https://nikhil0905.github.io/0xlunatic/](https://nikhil0905.github.io/0xlunatic/)

## About
This portfolio is built to showcase my journey, philosophy, skills, and projects as an **Aspiring Security Professional**. Built specifically for my pre-final year job hunt, it highlights my readiness for roles in Cloud Security, VAPT, and IAM. It features a unique "Jekyll & Hyde" dual-mode interface:

* **Executive View:** A premium "Tactical Sentinel" command center design featuring deep obsidian surfaces, bento-grid layouts, ambient blooms, and sharp borders built for recruiters and executives.
* **Root Access View:** A fully interactive, terminal-based hacker theme for engineers, featuring real commands (`help`, `whoami`, `skills`) and typing animations.

### Features
* **Aether Sentinel Design System:** Custom-built 5-tier surface hierarchy featuring noise textures, glowing gradients, and sharp UI elements for an immersive cybersecurity aesthetic.
* **Dual Theme Engine:** Instantly swap between professional and terminal modes without reloading.
* **Interactive Terminal:** An actual working pseudo-shell built in JavaScript that accepts inputs and processes standard commands.
* **Secure by Design:** The codebase is hardened with strict Content Security Policies (CSP), reverse tabnabbing protection (`rel="noopener noreferrer"`), enforced TLS, and robust DOM sanitization.
* **Premium Micro-Interactions:** Custom magnetic cursor effects, text scramble/decode animations, parallax background hexagons, staggered scroll reveals, and synchronized audio feedback.

## Tech Stack
* **HTML5** (Semantic Structure, SVG Elements)
* **CSS3** (Variables, CSS Grid Bento Layouts, Advanced Keyframe Animations, Grain/Noise Textures)
* **Vanilla JavaScript** (Audio Synthesis, Terminal Logic, Intersection Observers, Parallax Logic)
* No heavy frameworks. Designed to be lightweight and fast.
